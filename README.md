# 🍔 Sistema de Pedidos - Lanchonete Python

Este é um sistema de linha de comando (CLI) desenvolvido em Python para gerenciar pedidos de uma lanchonete. O projeto simula o fluxo real de um carrinho de compras, permitindo ao usuário interagir com um cardápio dinâmico.

O sistema foi desenvolvido como parte de uma avaliação acadêmica, com o objetivo de aplicar conceitos fundamentais de lógica de programação e estruturas de dados na prática.

## 🚀 Funcionalidades

* **Exibição do Cardápio:** Mostra os itens disponíveis com seus respectivos preços de forma dinâmica.
* **Adição de Itens:** Permite adicionar lanches e bebidas ao carrinho de compras utilizando índices numéricos automáticos.
* **Remoção Dinâmica:** Remove itens específicos do carrinho através do método `.pop()` e atualiza o valor total da comanda instantaneamente.
* **Validação de Dados:** Sistema protegido contra erros comuns, impedindo a remoção de itens caso o carrinho esteja vazio.
* **Cálculo da Comanda:** Atualização automática do valor total a cada inserção ou remoção de produto.

## 🛠️ Tecnologias Utilizadas

* **Python 3**
* **Dicionários (`dict`):** Utilizados para estruturar o cardápio (relação produto/preço).
* **Listas (`list`):** Utilizadas para armazenar os itens do pedido e gerenciar os índices dinamicamente.
* **Laços de Repetição (`while` e `for`):** Controle do menu principal e iteração sobre os dados.
* **Estruturas Condicionais (`if`, `elif`, `else`):** Validação das escolhas do usuário e tratamento de regras de negócio.
